Ez a Wordpress plugin létrehoz egy csokoládé esőt a weblapon, miközben opcionálisan megszólalhat Tay Zonday-től a Chocolate Rain című remekmű
EN: This is a Wordpress plugin that creates a chocolate rain on every site, while you also can listen the Tay Zonday classic, Chocolate Rain.

== Description ==

A plugin aktiválás után rögtön életbe lép és megkezdődik a csokoládé eső.

EN: After the plugin activation the rain starts falling.
